# 00 总览与快速开始

## 场景说明（我在做什么）
你要在 Windows11 本地把这个仓库跑起来，先完成：下载行情数据、跑一次回测（Backtesting）、再启动模拟盘（Dry-run）。

## 复制即用命令（Windows PowerShell）
```powershell
cd .\trading-strategies
```

```powershell
freqtrade download-data --userdir user_data --exchange binance --trading-mode futures --timeframes 1m 15m 1h 4h 1d --pairs BTC/USDT:USDT ETH/USDT:USDT SOL/USDT:USDT --timerange 20230101-
```

```powershell
freqtrade backtesting --config user_data/configs/S1_config_stub.json --strategy S1_HFT_Conservative_MicroTrend_Scalper --timerange 20230101-20240630 --export trades --backtest-filename user_data/backtest_results/S1_train.json
```

```powershell
freqtrade trade --config user_data/configs/S1_config_stub.json --strategy S1_HFT_Conservative_MicroTrend_Scalper --dry-run
```

## 参数解释（大白话）
- `--userdir user_data`：告诉 Freqtrade 你的策略和配置都在 `user_data`。
- `--trading-mode futures`：使用合约（Futures）模式。
- `--pairs BTC/USDT:USDT ...`：只下载/交易这几个币对，减少复杂度。
- `--timerange`：回测时间段，越长越能看稳定性。
- `--dry-run`：模拟盘，不会真实下单。

## 结果怎么看（成功/失败判断）
- 数据下载成功：终端出现下载完成，无红色报错。
- 回测成功：生成 `user_data/backtest_results/S1_train.json` 文件。
- Dry-run 成功：终端持续输出日志，并显示策略开始扫描信号。
