# 06 实盘前检查清单

## 场景说明（我在做什么）
你准备从模拟盘走向小资金实盘前，做最后风控确认。

## 复制即用命令（Windows PowerShell）
```powershell
python -m compileall user_data/strategies
```

```powershell
rg -n "shift\(-[0-9]+\)|TODO|FIXME|BUG|HACK" user_data/strategies
```

## 参数解释（大白话）
- `compileall`：检查策略是否有语法错误。
- `rg ...`：扫描是否存在未来函数风险和未完成标记。

## 结果怎么看（成功/失败判断）
- 成功：编译通过；扫描无命中（或只有可解释文本）。
- 失败：任何报错都先修复，不要上实盘。
