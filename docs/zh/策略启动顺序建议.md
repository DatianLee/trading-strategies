# 策略启动顺序建议（仅 Dry-run）

## 场景说明（我在做什么）
你是新手，想知道该先跑哪 3 个策略，并用什么“保守默认参数”。

## Top 3（新手优先）

### 1) S5_LFT_Conservative_MTF_TrendReversal
- 选择理由：
  - 回撤容忍低（保守层），更适合先建立信心。
  - 1h 低频，信号不乱跳，维护压力低。
- 保守默认参数（建议）：
  - `default_leverage: 1.0`
  - `max_open_trades: 2`
  - 交易对：`BTC/USDT:USDT`, `ETH/USDT:USDT`, `SOL/USDT:USDT`

### 2) S3_MFT_Conservative_TrendPullback
- 选择理由：
  - 趋势回撤策略，信号逻辑直观。
  - 比 1m 高频更容易看懂问题来源。
- 保守默认参数（建议）：
  - `default_leverage: 1.0`
  - `max_open_trades: 2`
  - 时间周期：`15m`

### 3) S1_HFT_Conservative_MicroTrend_Scalper
- 选择理由：
  - 属于保守组，但频率更高，可补充样本量。
  - 适合在前两个策略稳定后再加入。
- 保守默认参数（建议）：
  - `default_leverage: 1.0`
  - `max_open_trades: 2`
  - 时间周期：`1m`

## 复制即用命令（Windows PowerShell）
```powershell
freqtrade trade --config user_data/configs/S5_config_stub.json --strategy S5_LFT_Conservative_MTF_TrendReversal --dry-run
```

```powershell
freqtrade trade --config user_data/configs/S3_config_stub.json --strategy S3_MFT_Conservative_TrendPullback --dry-run
```

```powershell
freqtrade trade --config user_data/configs/S1_config_stub.json --strategy S1_HFT_Conservative_MicroTrend_Scalper --dry-run
```

## 参数解释（大白话）
- `default_leverage`：杠杆，先固定 1 倍最稳。
- `max_open_trades`：同时最多开几单，先从 2 单开始。

- 成功：7 天内没有触发硬停机，回撤可控，PF 稳定 > 1。

## 满足这些门槛才允许进入小资金实盘
1. 连续 7 天 Dry-run 运行不中断。
2. 组合 PF ≥ 1.15。
3. 最大回撤 ≤ 6%。
4. 最大连续亏损 ≤ 5。
5. 严格按保守配置（1x 杠杆 + 低并发）再跑 24 小时无异常。
