# 03 回测与结果解读

## 场景说明（我在做什么）
你要验证策略在历史数据上的表现，并看它有没有明显退化。

## 复制即用命令（Windows PowerShell）
```powershell
freqtrade backtesting --config user_data/configs/S5_config_stub.json --strategy S5_LFT_Conservative_MTF_TrendReversal --timerange 20230101-20240630 --export trades --backtest-filename user_data/backtest_results/S5_train.json
```

```powershell
freqtrade backtesting --config user_data/configs/S5_config_stub.json --strategy S5_LFT_Conservative_MTF_TrendReversal --timerange 20240701-20241231 --export trades --backtest-filename user_data/backtest_results/S5_test.json
```

```powershell
python scripts/validate_backtest_gates.py --strategy S5 --train user_data/backtest_results/S5_train.json --test user_data/backtest_results/S5_test.json --max-dd 0.10 --max-profit-factor-delta 0.55 --max-winrate-delta 0.10
```

## 参数解释（大白话）
- `train`：训练窗口，代表旧市场。
- `test`：测试窗口，代表新市场。
- `validate_backtest_gates.py`：自动判断“有没有退化太多”。
- `max-dd`：最大允许回撤（越小越稳）。

## 结果怎么看（成功/失败判断）
- 成功：门槛校验脚本输出通过（PASS）。
- 失败：任何一项超阈值都要重调参数或换策略。
