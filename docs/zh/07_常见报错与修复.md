# 07 常见报错与修复

## 场景说明（我在做什么）
你命令运行失败，需要快速定位并修复。

## 常见问题
1. **`freqtrade` 不是内部命令**
   - 修复：重新打开 PowerShell，执行 `py -3.11 -m pip install freqtrade`。
2. **策略找不到**
   - 修复：确认 `--strategy` 名称与类名完全一致（区分大小写）。
3. **数据不足无法回测**
   - 修复：先执行 `download-data` 并扩大 `--timerange`。
4. **交易所连接失败**
   - 修复：检查 API key、网络和交易所可用性。

## 复制即用命令（Windows PowerShell）
```powershell
freqtrade list-strategies --userdir user_data
```

```powershell
freqtrade download-data --userdir user_data --exchange binance --trading-mode futures --timeframes 1m 15m 1h 4h 1d --pairs BTC/USDT:USDT ETH/USDT:USDT SOL/USDT:USDT --timerange 20230101-
```

## 参数解释（大白话）
- 先用 `list-strategies` 排除“策略加载失败”。
- 再用 `download-data` 排除“历史数据不够”。

## 结果怎么看（成功/失败判断）
- 成功：策略可见 + 数据下载完成。
- 失败：按报错关键字逐条修复。

## 下一步做什么
修复后回到 `03_回测与结果解读.md` 重新跑。
