# 10分钟新手路线图（零基础）

> 目标：今天只做最短路径：安装 → 下载数据 → 回测最稳策略 → 启动模拟盘（Dry-run）。

## 第1步：安装 Freqtrade
```powershell
py -3.11 -m pip install freqtrade
```
看到什么代表成功：终端出现 `Successfully installed ...`。

失败时最可能修复：先执行 `winget install -e --id Python.Python.3.11` 安装 Python 3.11。

## 第2步：进入仓库
```powershell
cd .\trading-strategies
```
看到什么代表成功：命令行路径变成 `...\trading-strategies`。

失败时最可能修复：确认你当前目录下有 `trading-strategies` 文件夹；没有就先 `git clone` 到当前目录。

## 第3步：下载历史数据
```powershell
freqtrade download-data --userdir user_data --exchange binance --trading-mode futures --timeframes 1m 15m 1h 4h 1d --pairs BTC/USDT:USDT ETH/USDT:USDT SOL/USDT:USDT --timerange 20230101-
```
看到什么代表成功：输出里出现下载完成，不再报红错。

失败时最可能修复：检查网络，或重试一次（交易所偶发超时）。

## 第4步：回测最稳策略（S5）
```powershell
freqtrade backtesting --config user_data/configs/S5_config_stub.json --strategy S5_LFT_Conservative_MTF_TrendReversal --timerange 20230101-20240630 --export trades --backtest-filename user_data/backtest_results/S5_train.json
```
看到什么代表成功：生成 `user_data/backtest_results/S5_train.json`。

失败时最可能修复：先确认第3步数据已成功下载。

## 第5步：启动模拟盘（S5）
```powershell
freqtrade trade --config user_data/configs/S5_config_stub.json --strategy S5_LFT_Conservative_MTF_TrendReversal --dry-run
```
看到什么代表成功：持续输出日志，显示策略正在运行。

失败时最可能修复：若报交易所权限问题，检查 API key 是否开启合约读取/交易权限。
